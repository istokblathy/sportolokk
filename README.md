Alapinformációk
Projekt címe: Sportolók

Csapattagok: Balázs Gergő, Istók Ádám

Választott téma: Athletes – sportolók adatainak kezelése és megjelenítése

Adatmodell rövid leírása
A felhasználó megadhatja egy sportoló adatait (név, sportág, életkor, értékelés), amelyekből a rendszer egy strukturált adatsort hoz létre. Ezek az adatok HTML‑ben jelennek meg, táblázatos vagy kártyás formában, és a JavaScript gondoskodik a keresésről, szűrésről és a kedvencek kezeléséről.
